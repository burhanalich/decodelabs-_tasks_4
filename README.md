# General Knowledge Quiz

A simple Python command-line quiz game that tests the user's general knowledge with multiple-choice style questions. The program checks answers, keeps track of the score, and displays feedback based on the final result.

## Features

- Interactive quiz with multiple questions
- Instant feedback after each answer
- Calculates and displays the final score
- Motivational message based on performance
- Beginner-friendly Python project

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/decodelabs_tasks.git
```

2. Navigate to the project folder:

```bash
cd decodelabs_tasks
```

3. Run the program:

```bash
python general_knowledge_quiz.py
```
